| Question | Main deliverable |
|---|---|
| 1. Understanding Kamereo | [`kamereo/answers.md`](kamereo/answers.md) |
| 2. SQL — best & least selling categories | [`SQL/1_best_and_least_selling_categories.sql`](SQL/1_best_and_least_selling_categories.sql) |
| 3. Power BI — descriptive analysis | [`PowerBI/Descriptive_Analysis.md`](PowerBI/Descriptive_Analysis.md) + `PowerBI/top_50_fast_food_chains_presentation.pbip` |

## Structure

```
kamereo/
├── answers.md                  ← interview Q&A (main)
└── research/                   ← company research notes (facts, business model,

SQL/
└── 1_best_and_least_selling_categories.sql   ← main
PowerBI/
├── Descriptive_Analysis.md     ← written analysis of the deck (main)
```
