# Kamereo Interview Test

Three questions, three folders. The main deliverable sits at the top of each folder; supporting material lives in its `research/` subfolder.

| Question | Main deliverable |
|---|---|
| 1. SQL — best & least selling categories | [`SQL/1_best_and_least_selling_categories.sql`](SQL/1_best_and_least_selling_categories.sql) |
| 2. Power BI — descriptive analysis | [`PowerBI/2_descriptive_analysis.md`](PowerBI/2_descriptive_analysis.md) + `PowerBI/top_50_fast_food_chains_presentation.pbip` |
| 3. Understanding Kamereo | [`kamereo/3_understanding_kamereo_answers.md`](kamereo/3_understanding_kamereo_answers.md) |

## Structure

```
SQL/
└── 1_best_and_least_selling_categories.sql   ← main
PowerBI/
├── 2_descriptive_analysis.md   ← written analysis of the deck (main)
├── top_50_fast_food_chains_presentation.pbip  ← Power BI project (open in Desktop)
├── top_50_fast_food_chains_presentation.pbix  ← one-click view, data included
└── research/                   ← PRD, decision logs, house style rules
kamereo/
├── 3_understanding_kamereo_answers.md   ← interview Q&A (main)
└── research/                   ← company research notes (facts, business model,
                                   market, leadership, video notes, synthesis)
```
